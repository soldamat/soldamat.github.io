-- ATUALIZAR FOTO SITE SOLDAMAT EQUIPE -- 


no repositório website-equipe-workflow (github.com/soldamat/website-equipe-workflow),
tu vai em actions (github.com/soldamat/website-equipe-workflow/actions)
e seleciona na coluna lateral esquerda "Atualiza Equipe no site" (https://github.com/soldamat/website-equipe-workflow/actions/workflows/update.yml)

clica Run Workflow > Run Workflow

quando acabar, ele vai abrir um Pull Request no repositório https://github.com/soldamat/soldamat.github.io (o repositório é http://soldamat.github.io/)

aqui vai ter o .zip com o preview. Abrir onde foi alterado e checar e, depois, clicar no botão verde "Merge pull request"
